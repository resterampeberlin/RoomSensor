Purpose

This arduino application runs on en ESP8266, captures data from the sensors the one wire bus and sends it to a MySQL server. This data can be used to be displayed in nice graphs A typical application will be to capture temperature data in a room

Compatibility

This is program has been tested with

  • ESP8266
  • Maxim DS18B20 digital temperature sensor

Installation

Sensor

  1. Refer to sketch RoomSensor.sch for wiring (wire GPIO2 to 1-wire bus with 4,7K pullup)
  2. Create file confidential.h in which you define host, WLAN name and password (see ssid, password, host)
  3. Define locationId in Roomsensor.ino
  4. Compile and load to ESP8266
  5. Connect the sensor one by one (after a reset) and remember unique HW-Id using the serial monitor
  6. Modify sensorInfo with the remembered values
  7. Compile and load again. Now all sensor should be marked as "found"

Server

  1. Install MySQL/MariaDb and PHP on server
  2. Install jpgraph library on server (see http://jpgraph.net)
  3. Create datebase Smarthome (See SmartHome.sql)
  4. modify tables sensor and location
  5. Install server scripts
  6. Create file Confidential.php with user name and password for mysql server (see Smarthome.php)
